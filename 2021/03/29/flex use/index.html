flex布局 用法记录

flex container: 开启flex布局的元素

flex items: flex container里的直接子元素

开启方法: 设置元素的css属性 display: flex（块级元素）或 inline-flex（行内元素）

flex模型相关尺寸:

flex constainer的css属性：

flex-flow

flex-direction

flex-wrap

justify-content

flex items的css属性:

flex

flex-grow

flex-basis

flex-shrink

order

align-self
